Mutual Funds Analysis

In this project, I created an interactive dashboard from the data of Mutual funds Schemes.

Following are the key points of the project that I highlighted in the dashboard:

  1. Top performing Asset management companies
  2. Top performing schemes
  3. Top schemes by Size
  4. Returns from different Categories of Mutual Funds

The approach I followed to achieve these insights

  1. Created a data model by importing data from multiple sources.
  2. Used complex parameters to drill down in worksheet and customisation using filter and slicers.
  3. Created connections, join new tables and enable user driven parameters for visualizations.
